# Secure Steganography System

A Python desktop application that securely hides secret messages inside images using LSB steganography and password-based encryption.

## Features

- Hide secret messages inside images
- Extract hidden messages
- Password-protected encryption
- SHA-256 integrity verification
- Steganography detection
- Brute-force attack simulation
- Modern Tkinter GUI

## Technologies

- Python
- Tkinter
- Pillow
- Stegano
- Cryptography (Fernet)

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python steg_vapt.py
```
